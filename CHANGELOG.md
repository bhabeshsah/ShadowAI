# Changelog

All notable changes to the Shadow Voice Assistant project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-05-16

### Added
- Initial release of Shadow Voice Assistant
- Wake word detection using "Shadow"
- Speech recognition and text-to-speech functionality
- Integration with OpenRouter API for LLM responses
- Environment variable configuration
- Basic logging system
- Documentation and setup instructions
